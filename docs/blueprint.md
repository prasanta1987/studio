# **App Name**: Virtuoso Keys

## Core Features:

- Interactive 88-Key Keyboard: Display a responsive and visually appealing 88-key digital keyboard on the main page. Keys will visually highlight when pressed virtually or via MIDI input.
- MIDI Keyboard Connectivity: Enable connectivity with external MIDI keyboards. The app should detect MIDI input and highlight the corresponding keys on the virtual keyboard.
- Tone Selection: Provide a selection of diverse instrument tones (e.g., piano, strings, synth sounds).
- Rhythm Accompaniment: Offer various rhythm accompaniments spanning diverse musical genres, enhancing the practice and performance experience.
- Record and Playback: Implement recording functionality, enabling users to record their performances and play them back.

## Style Guidelines:

- Primary color: Deep blue (#293462) to evoke a sense of professionalism and sophistication.
- Background color: Light gray (#F0F0F0), providing a clean and unobtrusive backdrop.
- Accent color: Vibrant orange (#FF6B35) for interactive elements and key highlights.
- Body and headline font: 'Inter', a sans-serif font known for its modern, clean appearance.
- The keyboard will take up most of the screen with controls placed logically above and below the keyboard.
- Use flat, minimalist icons for controls like tone selection, rhythm selection and record/playback
- Use subtle animations for visual feedback when pressing keys and interacting with controls.
